# X Silly Feed

A Chrome extension that transforms X.com (Twitter) into a silly, sketch-inspired newspaper aesthetic using the experimental `border-shape` CSS property.

## Features
- **Organic Borders**: Uses CSS `border-shape` for hand-drawn-looking UI elements.
- **Silly Aesthetic**: Sketch-like typography and paper-textured backgrounds.
- **Pure CSS**: Highly performant implementation with minimal JS overhead.

## Tech Stack
- Manifest V3
- CSS (Experimental `border-shape`)
- JavaScript

## Setup
1. Clone the repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the project directory.
5. **Note**: Requires Chrome 147+ (or Canary with Experimental Web Platform features enabled) to see `border-shape` effects.
