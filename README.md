## 🧐 What is X Silly Feed?

**X Silly Feed** is a Chrome extension that transforms the rigid, corporate UI of X.com (formerly Twitter) into a whimsical, sketch-inspired newspaper aesthetic. 

By leveraging the cutting-edge (and experimental) CSS `border-shape` property, we replace the soul-crushing rectangles of social media with organic, hand-drawn wiggles. Combined with paper textures and vintage editorial typography, your doomscrolling experience becomes a lighthearted trip to a Victorian newsstand.

---

## Features 🚀

- **🎨 Organic Borders**: Real-time hand-drawn effects using experimental CSS `border-shape`. No bulky images, just pure math and ink.
- **📜 Newspaper Aesthetic**: Custom typography pairings (Cormorant Garamond & Libre Baskerville) and paper-grain textures.
- **⚡ Performance First**: Zero heavy libraries. Built with pure CSS and minimal JavaScript to ensure your feed stays fast.
- **🛡️ Privacy Centric**: All processing happens locally on your machine. We don't track you, we don't log you, we just make you smile.
- **🌑 Dark Mode Support**: Works beautifully in both light and dark themes (though we recommend light mode for the full newsprint feel!).

## Quickstart ⚡

Transform your timeline in under 60 seconds:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/tuliosousapro/x-silly-feed-ui.git
   ```

2. **Open Extensions**:
   Navigate to `chrome://extensions` in your browser.

3. **Enable Developer Mode**:
   Toggle the switch in the top-right corner.

4. **Load the Extension**:
   Click **"Load unpacked"** and select the folder you just cloned.

> [!IMPORTANT]
> **Experimental Features Required**: To see the `border-shape` effects, you need Chrome 147+ (Canary) or to enable the **"Experimental Web Platform features"** flag in `chrome://flags`.
