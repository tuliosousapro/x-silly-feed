# Release Notes - X Silly Feed

## Version 1.0 (2026-05-01)

### Added
- Initial release of the X Silly Feed theme.
- Support for experimental CSS `border-shape` property to create organic, hand-drawn borders on tweets, sidebars, and buttons.
- Newspaper-style typography overrides using handwritten fonts.
- Paper-textured background for the entire X.com UI.
- Popup toggle to enable/disable the theme on demand.
- Manifest V3 architecture with a persistent background service worker for state management.
- Hand-drawn SVG icon matching the silly, sketch aesthetic.

### Improved
- Added silly styling to specific Twitter/X modals and popups in the `#layers` container.
