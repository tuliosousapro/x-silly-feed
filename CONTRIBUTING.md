Contributing 🤝
Contributions are what make the internet silly! Whether it's fixing a CSS bug or adding a new sketchy font, PRs are welcome.

Fork the Project
Create your Feature Branch (git checkout -b feature/SillyFeature)
Commit your Changes (git commit -m 'Add some Sillyness')
Push to the Branch (git push origin feature/SillyFeature)
Open a Pull Request
