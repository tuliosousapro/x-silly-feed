Quickstart Guide ⚡

Transform your timeline in under 60 seconds!

1

Clone Repo

Grab the source code from GitHub.

git clone https://github.com/tuliosousapro/x-silly-feed.git
2

Extensions

Open chrome://extensions in your browser.

3

Dev Mode

Toggle "Developer mode" in the top-right corner.

4

Load Unpacked

Click "Load unpacked" and select the project folder.

Experimental Features Required!

To see the border-shape effects, you need Chrome 147+ (Canary) or enable the Experimental Web Platform features flag in chrome://flags.
